このメールは、お使いのネットワーク環境内で Slack の機能を正常にご利用いただくために、サン電子M2M のオーナーまたはプライマリーオーナーにお送りしています。
2023 年 5 月 9 日までにネットワークまたは IT チームの協力のもと、新しいルート証明書を slack-edge.com のインフラストラクチャにインストールするようにしてください。
具体的には、Let's Encrypt の「ISRG Root X1」証明書がインストールされ、信頼されていることを確認する必要があります。
この証明書は、Let's Encrypt（https://letsencrypt.org/certificates/）からダウンロードできます。
Slack に接続するすべてのクライアントに、この証明書がインストールされている必要があります。
今後も Slack の機能を正常にご利用いただくために必要な措置ですので、今後数か月以内にできるだけ早めにご対応ください。
このルート証明書がすでにインストールされ、信頼されている場合、現時点で必要な対応はありません。
ご質問がありましたら、このメールに返信していただくか、feedback@slack.com までご連絡ください。
